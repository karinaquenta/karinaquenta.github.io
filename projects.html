NewsFlow
Full Stack Project

Live Headlines from around the world!

A live news platform for users who want to be informed with the latest news. This efficient and personalized web application allows for users to save articles on their own page to view at their leisure. Users are able to browse specific topics or view live headlines around the world to be informed.

The web application was built using React, Typescript, Vite, for frontend, and used Flask, Python, Postgres, SQLAlchemy for APIs. RESTful API utilized was NewsAPI.

What's the Weather?
Interactive Weather Application

Check the weather in your area!

A weather application with functionalities for users to search a city, state, or zipcode to obtain live weather updates. This efficient and personalized web application allows for users to save articles on their own page to view at their leisure. Users are able to easily check the weather with a user-friendly application.

The weather application was built using Javascript, Bootstrap and RESTful APIs. RESTful API utilized was OpenWeather.

F1 Racer
Full Stack Project

Search F1 Racer Information!

A project for users who are F1 Racer fanatics to look at stats by year and round. Data pulls the following information: position of the racer, full name of the racer, nationality, sponsor, and total points for the year and round of race.

The web application was built using Javascript, Typescript, and RESTful API integration.
